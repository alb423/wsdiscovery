wsdiscovery

1. code generation by gSoap.

2. only implement local discovery.

3. Build command <br \>
   make<br \>

4. Execute method <br \>
   Server side <br \>
      ./wsdiscovery<br \>
   Client: <br \>
      ./ws-client 2  <br \>
